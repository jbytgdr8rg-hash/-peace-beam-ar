# Peace Beam Invaders v10.9

v10.8をベースに、ハート表示、停止後の再起動モード、上部HUDの視認性を修正した版です。

## v10.9変更内容

- ハート表示が縦に並んだり潰れたりしないよう、横並び固定の `lifeText` 表示に変更
- 2人プレイ時でもハートが横に収まるよう、文字幅・間隔・横スケールを調整
- 停止後に再起動すると2人プレイになるバグ対策として、選択中モードを `selectedMode` として保持
- 開始時・停止時に選択中モードを明示的に復元
- 上部HUDの透明度を下げ、ブラーを強めて読みやすく調整
- v10.8の序盤攻撃量抑制、v10.6のMENU式ポーズレイヤー、BGM、横向き対応、2人手認識修正は維持しています

## 公開方法

zipを解凍し、以下をリポジトリ直下へ置いてください。

- `index.html`
- `.nojekyll`
- `README.md`

GitHub Pages設定は以下です。

- Source: Deploy from a branch
- Branch: main
- Folder: /(root)
