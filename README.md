# Peace Beam Invaders v9

カメラで手を認識して遊ぶブラウザARゲームです。

## v9追加機能

- 敵セットを倒すたびにスピードレベルが1段階アップ
- 最大20段階
- 基本敵に加え、5種類の別デザイン・別挙動の敵
- レベル10と20に大型ボス
- ボスは10発当てないと倒れません
- レベル20ボスは雑魚を召喚します
- 爆弾演出を強化
- 親指爆弾は30秒クールダウン
- 2人プレイモード
  - 画面を左右に分割
  - 左がP1、右がP2
  - 各ゾーンに敵が出現
  - スコアが高い方が勝ち
  - パーで相手ゾーンに敵を追加
  - パー妨害は20秒クールダウン

## 公開方法

`index.html` と `.nojekyll` をリポジトリ直下に置いて、Settings → Pages から公開してください。

公開URLは以下の形式です。

```text
https://<GitHubユーザー名>.github.io/<リポジトリ名>/
```

`file://` ではMediaPipeのモデル読み込みが失敗しやすいです。
